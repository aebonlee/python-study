# PyMaster - 프로젝트 계획서

## 프로젝트 개요
- **프로젝트명**: PyMaster (파이마스터)
- **목적**: 파이썬 학습자를 위한 체계적이고 인터랙티브한 웹 기반 학습 플랫폼
- **대상 사용자**: 파이썬 입문자부터 데이터 분석까지 학습하고자 하는 사람
- **개발 기간**: 2026년 3월
- **GitHub**: https://github.com/aebonlee/python-study

## 핵심 기능

### 1. 단계별 학습 시스템
- **기초 (Basics)**: 변수, 자료형, 조건문, 반복문, 리스트, 함수 등 9개 레슨
- **중급 (Intermediate)**: OOP, 예외 처리, 파일 I/O, 정규표현식 등 8개 레슨
- **고급 (Advanced)**: 데코레이터, 제너레이터, 동시성 등 5개 레슨
- **응용 (Applied)**: NumPy, Pandas 데이터 분석 5개 레슨

### 2. 실습 코드 에디터
- 브라우저 내 파이썬 코드 에디터
- 라인 넘버 표시
- Tab 키 지원
- 코드 실행 시뮬레이션 (print, 변수, f-string, for loop)
- 예상 출력과 비교 기능
- 힌트 제공

### 3. 도장깨기 배지 시스템
- 23개 배지 (브론즈 5, 실버 6, 골드 7, 플래티넘 4)
- 자동 배지 획득 평가
- 배지 알림 팝업
- 배지 컬렉션 페이지

### 4. 퀴즈 센터
- 4개 단계별 퀴즈 (각 10문제)
- 타이머 기능
- 해설 제공
- 점수 기록 및 재도전

### 5. 진도 관리
- 레슨 완료 체크
- 단계별 진도율
- 연속 학습일 (스트릭)
- localStorage 기반 데이터 저장

## 비기능 요구사항
- 반응형 디자인 (데스크톱, 태블릿, 모바일)
- 다크 모드 지원
- 클라이언트 사이드 렌더링 (SPA)
- 빠른 초기 로딩 (Vite)
- GitHub Pages 배포
